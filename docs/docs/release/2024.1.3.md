## 基本功能

---

- 添加探索菜单连接

- 支持记录访问书籍历史

- 修复书籍文档标记格式问题
- 优化书籍详情加载状态
- 支持记录访问者 IP 等信息
- 优化编辑文档路由
- 支持阅读文档时跳转编辑文档
- 添加书籍状态展示
- 添加首页热门书籍
- 支持修改文档名称
- 支持用户书籍展示
